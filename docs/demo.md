# Button

### 安装



``` javascript
import { Button } from 'vant';

Vue.use(Button);
```



## 使用

### Type
:::demo 默认样式

```html
<mi-vant-button>确认</mi-vant-button>

<mi-vant-button>取消</mi-vant-button>

```
:::


:::demo 禁用

```html
<mi-vant-button disabled>确认</mi-vant-button>
```
